Piston Liner Numerical Generation

利用信号处理及统计算法数值生成发动机气缸表面

Developed statistical and fast Fourier transform algorithm to calculate key topological features from measured piston liner surfaces.

Developed algorithms based on signal processing and random theory to numerically generate liner surfaces, which can be used for piston design and optimization.

The generated liner surfaces have similar physical properties compared with real ones.

A windows based graphical user interface was developed including all the algorithms mentioned above. With the GUI, sponsors do not need to look into the complex codes, but simply click the buttons.

Technique used in this project: MATLAB, C++, Signal Processing, Statistics

Companies using my results: Shell, Volvo, Toyota, MTU, and etc.

For more details: Please refer to my graduate thesis or contact me directly.

  • Date: July 2013 - Sept. 2015
  • Adviser: Dr. Tian Tian
  • Category: Graduate Research
